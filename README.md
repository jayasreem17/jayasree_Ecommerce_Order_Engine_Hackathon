# Ecommerce Order Engine Hackathon

**Author:** Jayasree M  

A Java console application for managing an e-commerce order system. This project allows users to create items, add them to a basket, and complete purchases through a checkout process.

---

## Project Structure

- Basket.java - Handles items in the basket  
- Checkout.java - Processes the purchase  
- Item.java - Represents individual items  
- Purchase.java - Stores completed purchases  
- ShopApp.java - Main class to run the program  

---

## How to Run

1. Open a terminal or Git Bash in the project root folder.  
2. Compile all Java files:

```bash
javac *.java
Run the program:
java ShopApp
Follow the prompts in the console to interact with the application.
Features
Create new items with name and price
Add items to basket
Complete purchase and view purchased items
Notes
Make sure Java is installed on your system.
Keep all .java files in the same directory when compiling.
